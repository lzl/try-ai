## Two core files

1. `/app/page.tsx` is the frontend code, including the workflow editor and debugger
1. `/app/api/chat/route.ts` is the backend code, including the workflow engine

## How to run

1. set env (see `.env.example`)
1. `npm install`
1. `npm run dev`
